# Gestion des commandes clients (gest_commandes)

## Contexte
Dans un restaurant, les commandes sont traitées selon le principe FIFO. Une file chaînée est la structure la plus adaptée.

## Objectifs pédagogiques
- Implémenter une file (FIFO)
- Comprendre les files dynamiques
- Gérer les cas particuliers de suppression

## Travail demandé
- Prise de commande
- Préparation terminée (défilement)
- Affichage des commandes en attente
- Annulation d’une commande
- Calcul du temps d’attente moyen
- Gestion des commandes urgentes

## Contraintes
- Liste simplement chaînée
- C++
- Interface console ou graphique avec les lib Qt ou SFML ou SDL
